- **김지윤**
  - 목록 페이지(이달의차트), 모달(투표하기)
  - 전역 CSS, 리소스

  - 목록 페이지(조공리스트), 모달(후원하기)

  - 유저 플로우

  - 마이 페이지, 모달(크레딧부족)

## 🗂️ 프로젝트 구조

```
codeit-FE10-Fandom-K
├─ .github                  : GitHub 설정
├─ .husky                   : Husky 설정
├─ public                   : Non-Compile 파일
└─ src                      : Compile 파일
    ├─ apis                 : 통신 API
    ├─ assets               : 미디어
    ├─ components           : 공용 컴포넌트
    │   └─ layout           : 레이아웃
    ├─ constants            : 상수
    ├─ contexts             : 전역 상태
    ├─ hooks                : 커스텀 훅
    ├─ pages                : 페이지
    │   └─ ~Page            : 개별 페이지
    │       └─ components   : 페이지 컴포넌트
    ├─ styles               : 전역 CSS
    └─ utils                : 유틸리티 함수
```
